## App

### Running the web app for development

```bash
# install @angular/cli
yarn global add @angular/cli

# cd into app
cd app

# install node modules
yarn

# run dev server
ng serve --port 4200
```

## Api

### Running for web api development

```bash
# cd into api
cd api

# install node modules
yarn

# run via nodemon
yarn dev
```
