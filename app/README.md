## Usage / Running

### WebApp Locally

```bash
# install node modules
yarn # or npm install

# run dev server
ng serve --port 4200

# build for prod
ng build --prod
```

### WebApp in Overwolf

```bash
# install node modules
yarn # or npm install

# run Overwolf Reload Dev Task
gulp dev
```
